RelateIQ Alpha API

If you're using the older alpha API, please know that it will be deprecated. We plan on leaving it available as people transition and we build up the functionality of the beta API but build all new integrations based on this Beta version of the RelateIQ API.

Future Announcements

Functionality will be documented here (or linked from here) so please check back for updates. Larger changes may have an accompanying announcement, but this documentation should always be up to date. Please contact us if you notice anything confusing or missing.
