The RelateIQ uses HTTP status codes to describe errors. In general, 200 is for successful requests, 4xx are errors with requests, and 5xx are unexpected server errors.
On error requests, the API returns a JSON object with more details.

Error object attributes

statusCode

Same as the HTTP status code

errorMessage

An error message to help the developer understand the problem

userMessage

An error message to for the end-user. (Not always descriptive for the user)
